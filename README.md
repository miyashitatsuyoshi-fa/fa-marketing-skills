# ファーストアカウンティング マーケティングチーム Manus スキル集

このリポジトリは、ファーストアカウンティング株式会社のマーケティングチームが Manus で利用するスキル（再利用可能なAIエージェントの作業手順書）を管理するためのものです。

## スキル一覧

| スキル名 | 概要 | トリガーとなる依頼例 |
| :--- | :--- | :--- |
| [event-lp-seo-cro-audit](./event-lp-seo-cro-audit/SKILL.md) | BtoBイベント・セミナーLPの包括的SEO・CRO監査 | 「このLPのSEO監査をして」「セミナーページのCRO分析をして」 |

## スキルの使い方

### Manus でスキルを利用する方法

1. このリポジトリの URL を Manus のスキル設定に追加してください。
2. 対象のLPのURLを貼り付けて「SEO・CRO監査をして」と依頼するだけで、スキルが自動的に起動します。

### スキルの追加・更新

新しいスキルを追加する場合は、以下の手順に従ってください。

1. `skill-name/` ディレクトリを作成する
2. `SKILL.md` を作成し、YAML フロントマター（`name`, `description`）と手順を記述する
3. プルリクエストを作成してチームでレビューする
